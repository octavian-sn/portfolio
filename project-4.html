My Go To Do

myGoToDo is an app that keeps track of all of your todo's, and you can sort them by project or due date.

Live Link

Project Overview

myGoToDo was the first major project that tied together my HTML, CSS and JavaScript skills. One of the problems I had, was re-attaching methods back to my objects after retrieving them from the local storage, which I eventually did by re-creating them using the original constructor.

The biggest issue though was, the complexity of the application, which resulted in a lot of code, that was necessary to be as organized as possible in order to make something out of it. In order to help me with this issue, I used modules to separate pieces of code. This project took a little over two weeks.
