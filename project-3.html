BATTLESHIP

Battleship is a game app, in which you have to sink the enemy AI's ships before it sinks yours.

Live Link

Project Overview

Battleship was created using vanilla JavaScript, and Jest for unit testing. The project's goal was to practice OOP. Testing the individual objects, really helped with keeping the project on track, since it became quite complex at one point.

The project's complexity really helped me with practicing my pseudo-coding abilities since it became apparent from the start that it was not going to be an easy project.

The trickiest part was the drag & drop of the ships onto the pick board, which I eventually solved by having the pick board retrieve information through event listeners from the dragged pieces.
